Starting Project Steps:
    1- Open Server and Client inside WebProject in VS code
    2- In Client run {npm run dev} to start the site
    3- Open site on http://localhost:5173/
    4- Start the server to make client active using npm start command
    5- Open ClothingDilemma.exe inside the Builds folder to start the game (Or open Main scene inside Assets\_Dev)
